API RESTful - Truellque

Bienvenido a la API de Truellque

Esta API permite gestionar items y categorías para un sistema de trueques local.

Carga inicial de categorías (seed)

Si estás corriendo el proyecto por primera vez, podés cargar las categorías base ejecutando:

node seedCategories.js

Endpoints disponibles

  • GET /api/items — Devuelve todos los items disponibles.
  • GET /api/items/search?name=algo — Buscar items por nombre.
  • GET /api/categories — Lista todas las categorías registradas.
  • POST /api/items — Crear un nuevo ítem (requiere JSON en body).
  • PUT /api/items/:id — Actualizar un ítem existente (requiere JSON en body).
  • DELETE /api/items/:id — Eliminar un ítem.

Parámetros útiles

Filtros opcionales en las URL:

  • ?category=Servicios — Filtrar por categoría
  • ?available=true — Mostrar solo los disponibles
  • ?name=guitarra — Buscar por título
